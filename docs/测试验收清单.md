# 测试验收清单

本文用于阶段二 2.1 回归测试、手工验收和后续阶段变更检查。

## 1. 自动化测试命令

后端：

```bash
backend/.venv/bin/python -m pytest backend/tests
```

前端：

```bash
cd frontend
npm test
npm run build
```

## 2. 环境检查

```bash
bash scripts/check-env.sh
```

应确认：

- Python 版本满足 3.10+。
- Node.js 版本满足 20.19+。
- FFmpeg 可执行文件存在。
- 后端依赖可导入。
- keyring 后端可用。

如果默认 `python3` 低于 3.10：

```bash
PYTHON_BIN=python3.11 bash scripts/check-env.sh
```

## 3. 手工验收主路径

1. 运行 `bash scripts/dev.sh`。
2. 打开 `http://127.0.0.1:5173`。
3. 确认合规弹窗持续停留，不会自动消失。
4. 点击“同意并使用”。
5. 确认顶部状态条能展示 SQLite、yt-dlp、FFmpeg、钥匙串、合规确认、Cookie 状态。
6. 输入公开视频链接并解析。
7. 确认封面、标题、UP主、时长、视频流、音频流展示正常。
8. 勾选视频和音频，下载为自动合并后的 `mp4`。
9. 下载完成后确认视频有画面且有声音。
10. 点击打开目录按钮，确认能打开输出目录。
11. 点击清空记录，确认已完成任务从任务队列消失，本地文件仍保留。

## 4. 资源选择矩阵

| 场景 | 预期 |
|---|---|
| 仅勾选视频 | 前端或后端阻止提交，提示需要同时选择音频 |
| 勾选视频 + 音频 | 自动合并并输出有声音的视频文件 |
| 仅勾选音频，输出 `m4a` | 只下载音频，不下载视频 |
| 仅勾选音频，输出 `mp3` | 只下载音频，并通过 FFmpeg 转为 MP3 |
| 仅勾选封面 | 只下载封面图片 |
| 勾选音频 + 封面 | 输出音频和封面，不下载视频 |
| 勾选视频 + 音频 + 封面 | 输出视频相关文件和封面 |
| 三类资源都不勾选 | 禁止提交，提示至少选择一个资源 |
| 未导入 Cookie 时只解析到 360P/480P | 前端显示低清晰度提示 |
| 导入有效登录 Cookie 后重新解析 | 解析阶段使用 Cookie，清晰度列表按资源权限展示更高清晰度 |
| 解析到 DASH 纯视频流 | 视频流保留在清晰度列表，提交时仍要求音频并自动合并 |

## 5. 批量文本解析验收

| 场景 | 预期 |
|---|---|
| 多行粘贴 2-3 个公开视频链接 | 自动生成逐行状态、逐条解析、成功项可提交 |
| 逗号、空格或普通文本混合输入 | 能自动识别支持的链接或 ID，并跳过无关文本 |
| 输入超过 50 条有效链接 | 只保留前 50 条，并提示超出数量 |
| 原始输入重复 | 默认跳过重复项 |
| 不同原始输入解析到同一规范化 URL | 后解析项标记为重复并跳过 |
| 部分链接解析失败 | 失败行保留错误信息，成功行仍可提交 |
| 展开批量行调整资源 | 单行资源选择只影响该行提交 payload |
| 展开批量行调整视频输出格式 | 单行 `container` 只影响该行提交 payload |
| 批量行只选视频不选音频 | 禁止作为可提交项，避免无声视频 |
| 批量行填写自定义文件名 | 仅该行任务使用该自定义文件名 |
| 批量提交成功项 | 每个成功项创建普通任务，按同时下载任务数进入任务队列 |
| 同时下载任务数设置为 2 后提交 3 个任务 | 同时最多 2 个任务运行，其余保持排队 |

## 6. 自定义文件名验收

| 场景 | 预期 |
|---|---|
| 不填写自定义文件名 | 输出文件名基于解析标题生成 |
| 填写自定义文件名 | 输出文件名基于用户输入生成 |
| 输入包含非法文件名字符 | 后端自动过滤非法字符 |
| 目标文件已存在 | 自动追加序号，不覆盖原文件 |
| 同名音频和封面输出 | 自动补资源后缀，避免文件名冲突 |

## 7. 容器与格式

| 项目 | 当前支持 | 不支持 |
|---|---|---|
| 视频格式 | `mp4`、`mkv` | `flv` |
| 音频输出 | `m4a`、`mp3` | 其他音频格式 |
| 封面 | 原始图片下载 | 封面嵌入视频元数据 |

## 8. 任务队列验收

应覆盖：

- 新任务显示下载时间。
- 任务队列显示封面缩略图；不勾选下载封面时仍可展示缩略图。
- 点击解析结果标题或任务标题，会在新页面打开对应视频页面。
- 完成任务显示完成时间。
- 运行中任务可取消。
- 失败或取消任务可重试。
- 完成任务可打开目录。
- 清空记录只删除已完成、失败、取消任务。
- 清空记录不删除本地下载文件。
- WebSocket 进度能更新前端状态。
- 快速刷新页面后，任务列表仍能从 SQLite 恢复。

## 9. Cookie 验收

基础验收：

- 未配置 Cookie 时，Cookie 状态显示未配置。
- 保存 Cookie 后，状态显示已配置和脱敏摘要。
- 删除 Cookie 后，状态回到未配置。
- SQLite 不保存 Cookie 明文。

暂缓验收：

- 会员/Cookie 授权资源下载。
- 不同网络下的 412/403 风控场景。
